# SkillRise

Learning platform. The **Next.js** app is in [`web/`](web/).

## Develop locally

1. `cd web && npm install`
2. Copy `web/.env.example` to `web/.env.local` and configure (never commit secrets).
3. `npm run dev` → [http://localhost:3000](http://localhost:3000)
